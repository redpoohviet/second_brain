*   **사이즈/레이아웃:** 1080px x 1920px. 수직적 흐름도 구성.
*   **기술 요소:**
    *   **[Layout]** Observe(관찰) $\rightarrow$ Extract(추출) $\rightarrow$ Automate(자동화)의 3단계 박스 구조를 사용한다. 각 단계는 `--color-bg-secondary`로 처리하여 구분성을 높인다.

    *   **[Technical Hook]** 각 단계 박스(`Observe_Box`, `Extract_Box`)는 API 연동을 전제로 하므로, 클릭/터치 이벤트 리스너를 통해 다음 섹션으로의 스크롤 이동(Scroll-Jacking) 트리거 역할을 할 수 있도록 정의한다.

## ✨ B. 핵심 모듈별 기술 사양서 (Module Tech Specs for Shorts/Reels - 9:16 Ratio)
### 1. 모듈 A: 후킹 타이틀 카드 (Hooking Title Card)
*   **목표:** 시청자의 스크롤을 강제적으로 멈추게 하는 'Pain Point' 자극.
*   **사이즈/레이아웃:** 1080px (W) x 1920px (H). 모바일 최적화 필수.
*   **기술 요소:**
    *   **[Interaction]** 진입 시, 배경에 미세한 필름 노이즈 애니메이션 (`opacity: 0` to `opacity: 0.1`)을 0.5초 간격으로 적용한다. (CSS `@keyframes`).
    *   **[Animation]** 핵심 키워드(Hook)는 '좌측에서 오른쪽'으로 스케일하며 진입하는 모션 (`transform: scaleX(0)` to `scaleX(1)`)을 사용하고, 타이밍은 0.3초를 넘지 않도록 제어한다.
    *   **[Component]** 좌측 상단에 '질문' 형식의 문구를 배치하며, 이는 시청자가 공감할 수 있는 **Question Mark Placeholder** 역할을 한다.

### 2. 모듈 B: OEA 프로세스 플로우 인포그래픽 (OEA Process Flow)
*   **목표:** 복잡한 지식(AI 자동화 원리)을 단계별로 구조화하여 신뢰도를 확보한다.
*   **사이즈/레이아웃:** 1080px x 1920px. 수직적 흐름도 구성.
*   **기술 요소:**
    *   **[Layout]** Observe(관찰) $\rightarrow$ Extract(추출) $\rightarrow$ Automate(자동화)의 3단계 박스 구조를 사용한다. 각 단계는 `--color-bg-secondary`로 처리하여 구분성을 높인다.
    *   **[Interaction]** **데이터 로딩 시퀀스:** 초기 상태에서는 모든 단계가 비활성화된 '점선'으로 표시된다. 스크롤 또는 영상의 특정 시간(Time Code)에 도달할 때마다 다음 단계 박스가 `opacity: 0`에서 `opacity: 1`로 부드럽게 페이드 인 되며, 동시에 내부 콘텐츠(예시 데이터)가 로딩되는 듯한 **'데이터 스트리밍 애니메이션'**이 발생해야 한다.
    *   **[Technical Hook]** 각 단계 박스(`Observe_Box`, `Extract_Box`)는 API 연동을 전제로 하므로, 클릭/터치 이벤트 리스너를 통해 다음 섹션으로의 스크롤 이동(Scroll-Jacking) 트리거 역할을 할 수 있도록 정의한다.

### 3. 모듈 C: CTA/수익화 전환 배너 (Conversion Banner Module)
*   **목표:** 콘텐츠 시청을 완료한 사용자에게 실질적인 '다음 단계'를 제시하여 수익화 Funnel로 유도한다.
*   **사이즈/레이아웃:** 1080px x 900px (영상 하단에 오버레이되는 형태로 설계).
*   **기술 요소:**
    *   **[Visual]** 뉴트럴 어스톤 배경 위에, 은은하게 반짝이는 **'미래 기술의 느낌(Tech-Luminescence)'**을 주는 그래픽 요소를 배치한다.
    *   **[Animation]** CTA 버튼(`var(--color-hook-accent)` 사용)에 마우스 오버/터치 시 미세한 '펄스 효과(Pulse Effect)'를 적용하여 클릭 유도를 극대화한다.
    *   **[Copywriting Hook]** "이 지식을 자동화하려면? $\rightarrow$ **심층 가이드북 다운로드 (클릭)**"와 같이 구체적인 액션을 제시하는 톤을 유지해야 한다.
