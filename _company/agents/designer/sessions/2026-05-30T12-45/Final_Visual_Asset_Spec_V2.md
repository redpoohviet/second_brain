# 🖼️ Jinny`s Lab - 최종 비주얼 에셋 구현 명세서 (Tech Spec) V2.0

## 📌 1. 적용 범위 및 원칙
*   **적용 콘텐츠:** '비개발자의 유튜브 자동화 시스템, 지금이 시작입니다' 스크립트 전체.
*   **최우선 원칙:** 모든 비주얼은 `var(--color-*)` CSS 변수를 사용하고, 정적인 이미지가 아닌 **인터랙티브 컴포넌트**로 구현되어야 합니다.

## 📌 2. 핵심 에셋 목록 (Assets)
1.  썸네일: 시스템 안정화 스토리텔링 기반의 Before/After 구조.
2.  후킹 타이틀 카드: 스크롤 애니메이션 및 타입라이터 효과.
3.  프로세스 플로우: 빛나는 궤적(Glowing Trail)을 이용한 데이터 흐름 시각화.
4.  CTA 컴포넌트: 인터랙티브 호버(Hover) 기능 포함.
