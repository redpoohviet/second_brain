# 📖 Module 1: '기초 개념 이해' 컴포넌트 명세서 (V1.0)

## 🎯 1. 목표 및 사용처 [근거: Designer 개인 메모리 / 지난 의사결정 로그]
*   **목표:** 교육 로드맵의 가장 첫 단추를 채우는 시각적 요소로, 학습자가 '여기서 무엇을 배우고, 왜 배워야 하는지'에 대한 명확한 동기 부여(Motivation)와 구조화된 정보를 제공합니다.
*   **사용처:** 랜딩 페이지 또는 교육 과정 개요 웹페이지의 상단 섹션 (Hero/Intro Section).
*   **핵심 컨셉:** 아날로그적 신뢰감과 체계적인 학습 흐름을 동시에 전달하는 **'Modular & Minimalist Tech Card'** 구조.

## 🖥️ 2. 인터랙티브 컴포넌트 상세 스펙 (Tech Spec) [근거: Designer 검증된 지식]
*   **컴포넌트 이름:** Info-Tab Component (정보 카드 탭 그룹)
*   **구조:** 3개의 핵심 정보 영역을 탭(Tab) 형태로 분리하여, 한 번에 과부하되는 느낌 없이 정보를 단계적으로 습득하게 유도합니다.
    1.  [Tab 1] Pain Point Trigger: "왜 이 지식이 필요한가?" (문제 제기)
    2.  [Tab 2] Solution/Concept: "무엇을 배우는가? (핵심 개념)" (정보 제공)
    3.  [Tab 3] Outcome/CTA: "배우면 어떤 결과가 오는가?" (결과 및 행동 유도)
*   **상태 변화 로직:**
    *   **기본 상태 (Default):** 모든 탭의 내용은 `opacity: 0`에서 시작하며, 마우스 오버나 클릭 시 **0.4초 동안 부드럽게 페이드 인(Fade-In)**됩니다. (`transition-duration: 0.4s; transition-timing-function: ease-out;`)
    *   **활성화 상태 (Active):** 현재 선택된 탭은 `var(--color-bg-secondary)` 배경색을 가지며, 헤드라인(`h3`)에 미세한 하이라이트 효과(underline)가 애니메이션됩니다.

## 🎨 3. 디자인 시스템 적용 가이드 [근거: Designer 검증된 지식]
*   **전체 배경:** `var(--color-bg-primary)` (뉴트럴 어스톤 / `#DDC2B9`) 유지. 미세한 노이즈 레이어 오버레이 (`opacity: 0.1` radial-gradient) 적용 권장.
*   **폰트 계층 구조:**
    *   **섹션 제목 (H2):** Pretendard, Bold, `var(--color-text-dark)` / 크기: `var(--size-h2)`
    *   **탭 헤더 (Tab Header):** Pretendard, Medium, `var(--color-text-dark)`
    *   **본문 내용 (Body Text):** Pretendard, Regular, `var(--color-text-dark)` / 크기: `var(--size-body)`
    *   **강조 용어 (Keywords):** Courier New 또는 유사 모노 스페이스체. 배경색으로 `#E8DDCB`를 사용해 구분합니다.

## 📐 4. 레이아웃 및 컴포넌트 좌표 명세 (Mockup Guideline) [근거: Designer 개인 메모리]
*(좌표는 Mobile-First 기준으로 설계되었으며, 데스크톱에서는 1200px 너비의 그리드 시스템에 맞춰 확장됩니다.)*

**A. 헤더 영역 (섹션 상단)**
*   **요소:** `[H2: 핵심 메시지]` / 위치: Top Left (X: 40px, Y: 60px) / 크기: 80% 너비 (`var(--size-h2)`).
*   **보조 요소:** `[설명 문구]` / 위치: H2 아래 중앙. (Padding Bottom: 20px).

**B. 탭 그룹 영역 (핵심 정보)**
*   **컨테이너:** 전체 너비 90% (`var(--size-container)`), 좌우 Padding 50px.
*   **Tab Headers:** (3개) X 좌표는 동일하게 배치하되, 간격은 `1.2em`을 유지합니다. 활성화된 탭 아래에 4px 두께의 진한 차콜 그레이 라인(Border-Bottom)을 추가하여 강조합니다.
*   **Tab Content Area:** (선택된 탭 아래) 최대 높이 제한 없이 콘텐츠가 자연스럽게 펼쳐지도록 `overflow: auto`를 적용하고, 패딩은 상하좌우로 40px씩 부여합니다.
