# 🎯 최종 썸네일 에셋 배포 및 기술 검증 체크리스트 (V3.0)

**1. 전처리 및 검증 단계 (Pre-Deployment)**
*   [ ] **CSS 변수 검토:** 모든 색상 코드와 크기가 `var(--color-bg-primary)` 등 CSS 변수로 정의되었는지 최종 확인.

*   [ ] **애니메이션 타이밍 동기화:** 후킹 제목 카드의 '반짝임' 효과가 영상 시작 3초 지점에서 정확히 발동하는지 키프레임을 재확인.

**2. 플랫폼별 구현 스텝 (Implementation Flow)**
*   **YouTube (Desktop/Mobile):**
    *   [ ] **A/B 테스트 로직 통합:** 썸네일 클릭 시, 내부적으로 어떤 가설을 적용했는지 추적할 수 있는 UTM 파라미터를 반드시 추가한다. (KPI 측정 필수)
    *   [ ] **배경 노이즈 레이어:** 배경에 아날로그 필름의 미세한 노이즈(Opacity 0.1)가 적용되었는지 검증하고, 영상 루프와 자연스럽게 연결되도록 처리한다.
*   **Instagram Reels (Mobile):**
    *   [ ] **9:16 비율 최적화:** 모바일 환경에 맞춰 OEA 다이어그램의 좌우 폭이 줄어들 때 정보가 압축되지 않고, 중앙 정렬되어 보이도록 리플로우(Reflow) 처리를 적용한다.
    *   [ ] **캡션 오버레이:** 썸네일 하단에는 플랫폼 고유의 CTA (예: "프로필 링크 클릭!")를 강제적으로 배치하여 유도성을 높인다.
