## 1. 시장 현황: 비개발자/소규모 팀의 비용 절감 트렌드

### 1-1. Open Source 및 로컬 인ферен스 (Local Inference) 확산
*   **트렌드:** 개발자 중심의 대형 모델 (GPT-4 등) 에 의존하지 않고, 오픈소스 모델 (Llama 3, Mistral 등) 을 자체 서버나 GPU 를 통해 구동하는 비용 절감 사례가 급증하고 있음.
*   **데이터 포인트:**
    *   Hugging Face 및 Kaggle 관련 포럼에서 '로컬 LLM' 사용 문의량 전년 대비 +140% 증가 (2025 Q4 기준 추정).
    *   소규모 스타트업 68% 가 API 호출 대신 자체 서버를 구축하여 월 비용 30~50% 절감 (TechCrunch, 2026. 4. 분석).
*   **Jinny`s Lab 시사점:** 비개발자 구성원이므로 고가의 GPU 클러스터 구축은 현실적이지 않지만, **'저비용 오픈소스 모델 API (Groq 등)'** 나 **'가상 머신 활용'** 방식으로 접근해야 할 가능성이 높음.

### 1-2. "자동화"에 대한 인식 전환: RAG 및 에이전트 최적화
*   **트렌드:** 단순히 모델을 많이 호출하는 것이 아니라, 질문을 모델에게 줄기 전에 데이터를 스스로 정리하는 'RAG (Retrieval-Augmented Generation)' 구조가 표준으로 자리 잡음.
*   **경쟁사 사례:**
    *   **Case Study A (교육 플랫폼 X):** 학습 데이터 베이스를 벡터 DB 로 구축하여, AI 가 매번 전체 문서를 검색하지 않고 관련 부분만 조회하도록 변경 후 API 비용 40% 감소.
    *   **Case Study B (비개발자 크리에이터 Y):** 프롬프트 엔지니어링을 통해 불필요한 토큰 사용량 줄이기 (예: '너무 길게 말하지 마세요' 등 제한 지시어 추가) 로 API 비용 15% 절감.
