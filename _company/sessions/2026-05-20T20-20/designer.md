CEO님의 지시는 현재까지의 모든 기획(Script/Research)을 실제 개발 단계에 투입할 수 있는 '구체적인 기술 명세서'로 완성하라는 의미입니다. 단순한 디자인 브리프가 아닌, **"이대로 코딩하면 동작하는 애니메이션 설계도"**를 제출해야 합니다.

## 🛠️ 콘텐츠 기반 인터랙티브 시스템 컴포넌트 기술 사양서 (Tech Spec V2.0)

**프로젝트:** Jinny`s Lab | 유튜브 쇼츠 (50~60초)
**테마:** 시스템적 구조 오류 진단 및 재설계
**핵심 원칙:** 정지 이미지 $\rightarrow$ 상호작용하는 흐름도(Flowchart/Data Visualization)

### 🖼️ Keyframe 1: [HOOK] 경고 - 스킬의 유효 기간 만료 (0s~3s)
*   **목표:** 시청자에게 강한 불안감과 즉각적인 주의를 환기시킨다.
*   **시각적 변화:** 화면 전체가 갑작스러운 '시스템 경고' 모드로 전환된다.

| 요소 | Tech Spec / CSS Variables | 애니메이션 로직 (Timing & Function) | 필수 에셋/소스 |
| :--- | :--- | :--- | :--- |
| **배경** | `var(--color-bg-primary)` 위에 붉은색 경고 오버레이 (`rgba(255, 0, 0, 0.1)`) 적용. | 초기 로드 시 (0s): 강한 깜빡임 애니메이션 (`animation: flash 0.3s infinite alternate;`). 배경에 미세 노이즈 레이어 유지. | - |
| **메인 카피** | "당신이 지금 배우는 스킬, 5년 뒤 사라질 수 있습니다." (H1/Hook) | 0.2초 지연 후 (`delay: 0.2s;`) `scale(1.0)`에서 `scale(1.2)`로 확대되며 강하게 등장. 글자별 타이핑 효과(`typing effect`). | **폰트:** *Brush Script* (또는 유사한 손글씨체). |
| **보조 요소** | 경고 아이콘 (`!`) + 시스템 코드 블록 형태의 텍스트 ("WARNING: SKILL DEPRECATION") | `opacity`를 0에서 1로 0.5초에 걸쳐 급격히 상승. 코딩 서체를 사용하여 기술적 느낌 강조. | **폰트:** *Courier New* (모노 스페이스). |
| **상호작용 로직** | 초기에는 비활성 상태(`disabled`). 사용자가 영상을 볼 때만 `is_active: true`가 되어 애니메이션이 작동해야 함. | - | - |

### 📊 Keyframe 2: [Problem A] Skill Mismatch Crisis (8s~25s)
*   **목표:** AI 기술 발전과 교육 시스템의 괴리를 데이터 그래프로 시각화한다.
*   **시각적 변화:** 두 개의 독립적인 성장 곡선이 그려지며, 그 간극(Gap)을 강조하는 인터랙션이 발생한다.

| 요소 | Tech Spec / CSS Variables | 애니메이션 로직 (Timing & Function) | 필수 에셋/소스 |
| :--- | :--- | :--- | :--- |
| **배경** | `var(--color-bg-secondary)`로 전환. 좌우 구조 분할 시작. | 배경은 부드러운 페이드 인 (`transition: background 1s ease;`). | - |
| **좌측 (AI)** | 제목: "AI 기술 발전 곡선" / 그래프 라인 A | **진행:** Time(X축)에 따라 가파르고 폭발적으로 증가하는 곡선(`cubic-bezier(...)`)이 애니메이션으로 그려진다. 0.5초 간격으로 점들이 찍히며 '성장'을 표현. | **그래프 에셋:** 급상승형 데이터 포인트 이미지 (SVG 또는 Canvas). |
| **우측 (교육)** | 제목: "기존 교육 시스템 변화 곡선" / 그래프 라인 B | **진행:** 완만하고 느린 기울기로 거의 수평에 가깝게 증가하는 곡선이 그려진다. AI와 대조되도록 움직임의 '느림'을 강조. | **그래프 에셋:** 완만한 상승형 데이터 포인트 이미지 (SVG 또는 Canvas). |
| **핵심 컴포넌트** | Gap Area: 두 라인 사이의 공간. | 1.5초 지점부터 Gap 영역에 어둡고 불안한 색상(예: `#B08C7A`)으로 음영 처리되며, 경고 문구("MISMATCH")가 깜빡인다 (`blink` 효과). | **텍스트:** "Systemic Lag" (모노 스페이스체). |

### 🌍 Keyframe 3: [Problem B] Regulatory Lag - 복잡한 규제 미로 (25s~40s)
*   **목표:** 국경 이동의 자유와 이를 담아내지 못하는 법적 프레임워크의 모호함을 시각화한다.
*   **시각적 변화:** 명확해야 할 개념들이 복잡하게 얽힌 '미로' 구조로 표현된다.

| 요소 | Tech Spec / CSS Variables | 애니메이션 로직 (Timing & Function) | 필수 에셋/소스 |
| :--- | :--- | :--- | :--- |
| **배경** | `var(--color-bg-secondary)` 유지. 중앙에 복잡한 노드(Node)들이 배치된다. | - | - |
| **핵심 노드 (A)** | "소득세 규정", "비자 만료일", "자산 이전 법규" 등 3개 이상의 키워드 블록. | 처음에는 독립적으로 나타나다가, 연결선(Line)을 통해 중앙에서 무질서하게 합쳐지며 복잡성을 증가시킨다 (Spaghetti Diagram 효과). | **아이콘:** 국가 경계/법률 관련 아이콘 세트. |
| **연결선** | 노드들을 잇는 흐름도 라인. | 이 연결선들이 마치 실타래처럼(Tangled) 휘감기며, 특정 지점('사각지대')에서 끊어지는 애니메이션을 구현한다. `transition-property: transform;`를 사용하여 유연한 움직임 강조. | **애니메이션:** 플로우차트 라인 (SVG Path). |
| **강조 컴포넌트** | '규제의 사각지대' 텍스트 블록. | 모든 노드가 최고조의 복잡성을 이룬 후, 중앙에 위치하며 주변 노드들을 흡수하는 듯한 연출로 부각된다. (`glow` 효과). | - |

### ✨ Keyframe 4: [Aha! Moment] 패러다임 전환 (40s~55s)
*   **목표:** 시선을 '스킬'에서 '시스템적 사고'로 강제 이동시키며 깨달음의 순간을 연출한다.
*   **시각적 변화:** 혼란스러웠던 좌우 구조가 명확한 3단계 플로우차트로 재정렬된다.

| 요소 | Tech Spec / CSS Variables | 애니메이션 로직 (Timing & Function) | 필수 에셋/소스 |
| :--- | :--- | :--- | :--- |
| **배경** | `var(--color-bg-primary)`로 부드럽게 복귀. 평온하고 명확한 느낌 강조. | 배경이 '시스템 재설정(Reset)'되며 톤앤매너가 급격히 전환됨을 시각적으로 알린다. | - |
| **구조화 컴포넌트** | 중앙에 3단계 플로우차트를 배치: Observe $\rightarrow$ Extract $\rightarrow$ Automate (OEA). | 이 세 개의 박스가 좌측에서 오른쪽으로 순서대로 `Staggered Fade-In` 애니메이션을 통해 등장한다. 각 단계별로 간격(Space)이 명확하게 분리됨을 강조. | **애셋:** OEA 프로세스를 나타내는 3개의 직사각형 컴포넌트. |
| **강조 요소** | '시스템적 사고'라는 핵심 키워드. | 이 문구는 다른 모든 요소보다 크고, 밝은 색상(예: 골드 계열)으로 강조되며 화면 중앙에 가장 먼저 등장하여 시선을 고정시킨다. (Focus Point). | **폰트:** *Pretendard* (가장 명확하고 모던한 느낌). |
| **전환 로직** | 이전 Keyframe 2, 3의 복잡성이 이 3단계 플로우차트를 향해 수렴(Convergence)하는 듯한 시각적 움직임을 연출. | - | - |

### ✅ Keyframe 5: [CTA] 명확한 다음 단계 제시 (55s~60s)
*   **목표:** 시청자의 행동을 유도하고, 채널 구독/댓글 참여를 요청한다.
*   **시각적 변화:** 모든 정보가 정리되고, 질문형 CTA에 집중하는 간결함으로 마무리된다.

| 요소 | Tech Spec / CSS Variables | 애니메이션 로직 (Timing & Function) | 필수 에셋/소스 |
| :--- | :--- | :--- | :--- |
| **배경** | `var(--color-bg-secondary)`를 사용하여 부드럽게 마무리한다. | - | - |
| **메시지 블록** | "더 이상 시스템의 오류로 시간을 낭비하지 마세요." (H2) | 화면 상단에서 부드럽게 슬라이드 인하며, 진중한 느낌으로 메시지를 전달한다. | - |
| **CTA 질문** | "당신은 어떤 '시스템적 지식'을 쌓아 올리시겠습니까?" (질문형 강조) | 가장 크고 명확하게 배치되며, 시청자에게 생각할 여유를 주는 템포로 등장. | - |
| **참여 유도 장치** | "댓글에 당신의 현재 고민(Pain Point)을 남겨주세요." / 구독 버튼 모방 인터랙션. | 댓글창과 연결된 듯한 UI 요소를 배치하고, 마우스 오버 시 활성화되는 애니메이션 효과를 부여하여 참여를 '유도'한다. | **UI 에셋:** 깔끔한 커서와 입력 필드 형태의 컴포넌트. |
