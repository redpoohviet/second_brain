## 1. Executive Summary (요약)
본 기획안은 베트남인 학습자가 겪는 언어 장벽, 시간 부족 등 **Pain Point**에 집중하여 '쇼츠 형식'으로 해결책을 제시하는 콘텐츠 전략을 다룹니다. 자동화된 숏폼 생성기를 통해 일관성 있고 고품질의 콘텐츠를 제공하며, 알고리즘 친화적인 후킹과 시각적 자극 요소를 결합한 실행 가능한 로드맵을 제공합니다.

## 2. 타겟 청중 인사이트 (Target Audience Insights)

### 🎯 주요 타깃층
1.  **베트남 거주/체류 베트남인**: 현지 생활, 직장, 비즈니스에서 중국어 능력 필요성 증가.
2.  **해외 거주 중인 한국인**: 중국어 학습 필요성 + 문화적 친숙도 활용 가능 (참고).

### 💡 Pain Point & 니즈 분석
| Pain Point | 데이터 기반 니즈 요약 | 콘텐츠 전략 방향 |
| :--- | :--- | :--- |
| **시간 부족** | 하루 30 분 내외의 집중력 유지 어려움 | **5~15 초 단위 단편 정보** (핵심 문법/단어) |
| **문법 복잡성** | 중국어 성조, 입체적인 발음 어려움 | **시각적 보조 (텍스트/이미지)** + **발음 가이드** |
| **실전 적용 부재** | 시험 대비보다는 실생활 회화 위주 선호 | **상황극/예시 대본 중심**의 콘텐츠 구성 |
| **동기 부여 부족** | 지속성 유지 어려움 | **도전 과제형 (Challenge)** 및 **성장実感** 강조 |

## 3. 콘텐츠 형식 분석 (Shorts Format Analysis)

### 🎬 자동 생성형 쇼츠 전략
비개발자가 쉽게 구현할 수 있는 **'텍스트 중심 + 시각 효과'** 접근법 권장.

| 전략 유형 | 특징 | 추천도 | 예시 |
| :--- | :--- | :--- | :--- |
| **A. 텍스트 기반 (Text-to-Speech)** | 중국어 발음과 한글 해설 동시 표출, 배경음악 활용 | ⭐⭐⭐⭐⭐ | "중국어 '마마'는 이렇게 말해요 🇨🇳" |
| **B. 시각적 자극 (Visual Hook)** | 충격적인 통계/이미지 + 텍스트 오버레이 | ⭐⭐⭐⭐ | "오늘은 중국어 실수를 10 개만 고칩니다!" |
| **C. 자동 생성 시나리오** | AI 가 자동으로 스토리보드 작성, 이미지 생성, 자막 추가 | ⭐⭐⭐⭐ | 하루 3 편 이상 대량 업로드 가능 전략 |

### 🛠️ 자동화 도구 연동 가이드 (코다리/레오 담당)
- **영상 생성**: Pictory / InVideo AI 등 템플릿 기반 툴 활용.
- **음성**: Azure TTS 또는 ElevenLabs (중국어 목소리 선택).
- **자막**: 자막 자동 인식 + 한글 병기 기능 필수 적용.
- **업로드**: 유튜브 API 를 통한 자동 업로드 스케줄링 설정.

## 4. 경쟁 채널 트렌드 분석 (Competitor Trends)

### 🔍 주요 특징
1.  **메인 키워드**: `중국어 회화`, `중국어 문법`, `중국 여행 영어/중국어`
2.  **포맷**: 짧은 문장 + 발음 교정 + 상황별 예문 중심.
3.  **성공 요인**: '실수 교정' 콘텐츠와 '생활 속 중국어' 위주.

### 📊 경쟁사 분석 데이터 (가상 데이터 기반)
| 채널 유형 | 주요 콘텐츠 | CTR 전략 | 우리 전략 차별화 포인트 |
| :--- | :--- | :--- | :--- |
| **강사 중심** | 긴 설명, 발음 교정 | 신뢰도 높음 | 자동화 + 텍스트 시각화 결합 (비개발자 접근성) |
| **여행 가이드** | 여행 팁, 식당 주문 | 실용성 강조 | '비즈니스/직장' 용어 추가 (현실적 니즈 반영) |
| **트렌드 기반** | 최신 중국 드라마 대본 | 유행성 활용 | 데이터 기반 키워드 선정 (SEO 최적화) |

## 5. 데이터 기반 기획 요소 도출 (Key Takeaways)

### 📈 트렌드 키워드 (SEO 구조화)
- **핵심**: `중국어 회화`, `중국어 문법`, `중국어 발음`
- **롱테일**: `중국어 실수 교정`, `중국어 일상회화`, `중국어 비즈니스 영어`
- **베트남어 키워드**: `Học tiếng Trung`, `Ngữ pháp Trung Quốc`, `Từ vựng tiếng Trung`

### ⚡ 후킹 포인트 (First 3 Seconds)
1.  **충격적 문장**: "이 중국어 실수, 절대 하지 마세요!"
2.  **시각적 결과물**: "오늘 5 분 만에 중국어 회화 1 단계 완성"
3.  **문제 제시**: "중국어 발음이 왜 틀릴까요? (오디오 + 텍스트 비교)"

### 🎨 시각 시스템 (Designer 협업 필수)
- **메인 컬러**: 중국어 학습의 '신뢰감'을 주는 파란색/푸른색 계열 또는 브랜드 톤.
- **텍스트 스타일**: 가독성 높은 고대비 폰트 사용 (중국어 한자 + 한글 병기).
- **썸네일 전략**: 텍스트 기반 CTR 최적화 (예: "실수 교정 3 가지", "초보자 필수").

## 6. 1 차 콘텐츠 아이디어 (Top 5 Trends)

### 📝 트렌드 메모 (가상 데이터 기반 제안)
1.  **베트남인 중국어 실수 Top 5**: 일상에서 자주 하는 발음/문법 실수 교정.
2.  **중국 식당 주문 필수 문구**: 현지 생활에 즉시 활용 가능한 회화집 중점.
3.  **중국어 성조 10 초 만에 기억하기**: 시각적 보조 (손짓) 를 활용한 성조 학습법.
4.  **베트남어-중국어 비교 학습**: 언어 유사성 활용하여 쉬운 단어 암기.
5.  **중국 드라마 대본 속 숨은 문법**: 드라마 클립 + 텍스트 해설 결합.

### 🧠 핵심 전략: '시스템 오류 (SEI)' 적용
- **오류 지표**: 학습자의 실수율 또는 시간 낭비량.
- **해결책**: 자동화된 콘텐츠가 제공하는 '시각적/청각적 교정'.
- **클라이맥스**: "이제부터는 중국어 실수가 줄어드는 시스템입니다."
