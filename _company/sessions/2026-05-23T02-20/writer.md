## 1. 콘텐츠 시퀀스 기본 구조 (The Golden Loop)

모든 쇼츠는 **[후크 (Hook)]** → **[학습지점 (Learning Point)]** → **[실생활 활용 (Application)]** 순서로 구성됩니다. 이는 '노출형' 특성을 반영한 최적의 흐름입니다.

| 단계 | 시간대 (가이드) | 핵심 기능 | 비고 |
| :--- | :--- | :--- | :--- |
| **🎣 후크 (Hook)** | 0s ~ 3s | 시청자 이탈 방지<br>(통계/질문/충격 이미지) | 텍스트 오버레이 필수, BGM 페이드 인 |
| **📚 학습지점** | 3s ~ 15s | 핵심 내용 전달<br>(단어/문법/표현) | 자막 타이밍 엄수, 시각적 강조 (하이라이트) |
| **💡 실생활 활용** | 15s ~ 20s | 상황 적용 예시<br>(대화와 맥락 제시) | 실제 발음 가이드 또는 상황 이미지 삽입 |

## 2. 핵심 카피라이팅 포맷 5 가지 유형 (Script Templates)

### 🟢 유형 A: 문법/표현 교정형 (Grammar Correction)
**컨셉**: 자주 실수하는 표현을 지적하고 올바른 발음/문법을 제시.
**적용 시나리오**: "안녕하세요" vs "닝하오", 동사 위치 교정 등

| 시간대 | 화면 (Visual) | 자막 (Text Overlay) | 내레이션/자막 텍스트 (Copy) |
| :--- | :--- | :--- | :--- |
| **0s-3s** | ❌ 빨간색 체크표 + 질문 표출<br>"중국어 실수 중?" | `❌ "안녕하세요"`<br>`😱 이건 틀렸어요!` | **"중국어 '안녕하세요' 이렇게 말하면 큰일 납니다!"** |
| **3s-15s** | ✅ 녹색 체크표 + 발음 가이드<br>문법 구조 시각화 | `닝하오 (Nǐ hǎo)`<br>`[동사+목적어 구조]` | **"올바른 표현은 '닝 하오'입니다.<br>중국어는 동사가 문장 전체를 지배합니다."** |
| **15s-20s** | 🇨🇳 상황 이미지 (점심시간)<br>"네, 잘 먹습니다" 예시 | `점심 메뉴`<br>`난이오 (Nán yào chī)` | **"실제로는 이렇게 씁니다:<br>'난이오' (난은 밥을) '난이오'(먹어요)."</** |
| **CTA** | 👉 구독 버튼 + 다음 영상 제목<br>"다음엔 이 실수!" | `💡 다음엔 어떤 실수나요?` | **"구독하시면 매일 이런 꿀팁 알려드립니다! 👇"** |

---

### 🔵 유형 B: 단어 암기/비주얼 학습형 (Vocabulary Flash)
**컨셉**: 시각적 자극을 통해 단어를 빠르게 기억하게 함.
**적용 시나리오**: 과일, 동물, 일상 용품 등

| 시간대 | 화면 (Visual) | 자막 (Text Overlay) | 내레이션/자막 텍스트 (Copy) |
| :--- | :--- | :--- | :--- |
| **0s-3s** | 🇨🇳 중국어 표기 + 발음<br>"이게 뭐예요?" | `🇨🇳这是什么？`<br>`(네이버?)` | **"중국어 '이게 뭐예요?'를 이렇게 읽습니다!"** |
| **3s-15s** | 🍎 과일 이미지 + 단어 카드<br>한글 발음 가이드 | `사과 (Píng guǒ)`<br>`🇨🇳 平果` | **"사과: '핑 과오'라고 읽어요.<br>'핑'(평) 은 평평하다는 뜻이에요."** |
| **15s-20s** | 🛒 마트 상황 + 쇼핑 카트 이미지<br>"사과 주세요" 예시 | `사과 주세요`<br>`Yào yī piàn pín guǒ` | **"마트에서 이렇게 주문하세요:<br>'와 (요)' 하나 '핑 과오'(사과)"** |
| **CTA** | 👉 단어장 다운로드 링크<br>"오늘 단어 복습!" | `💾 단어장 저장해두세요!` | **"복습할 단어가 필요하시면 구독 주세요! 👇"** |

---

### 🟡 유형 C: 속담/성격 학습형 (Idiom & Culture)
**컨셉**: 중국 문화와 관련된 유익한 표현을 통해 언어적 통찰 제공.
**적용 시나리오**: '금방' 관련 속담, 수양 등

| 시간대 | 화면 (Visual) | 자막 (Text Overlay) | 내레이션/자막 텍스트 (Copy) |
| :--- | :--- | :--- | :--- |
| **0s-3s** | 🧠 뇌 이미지 + 질문<br>"중국어 속담은?" | `🇨🇳 속담을 아세요?`<br>`"금방"` | **"중국인들은 '금방'이라고 하지 않습니다!"** |
| **3s-15s** | 🐢 거북이 이미지 + 속담 카드<br>"급하지 마세요" | `慢慢来 (Màn màn lái)`<br>`천천히 하세요` | **"중국인들은 '만 만 라이'(천천히) 라고 말합니다.<br>속담: '천리 길도 한 걸음씩'입니다."** |
| **15s-20s** | 🇨🇳 직장/학교 상황<br>"급하게 할게요" vs "천천히" | `职场 (직장)`<br>`慢慢来` | **"직장에서 '급하게 하세요'라고 하면 안 됩니다.<br>오히려 '천천히'가 더 좋습니다."** |
| **CTA** | 👉 중국어 문화 채널 구독<br>"다음엔 속담 3 개!" | `📚 중국어 문화 채널 구독` | **"중국인의 사고방식도 배워보세요! 👇"** |

---

### 🟣 유형 D: 발음 훈련형 (Pronunciation Drill)
**컨셉**: 성조와 발음을 정확히 익히는 연습.
**적용 시나리오**: 'Sh' vs 'S', 'Zh' vs 'Z' 등

| 시간대 | 화면 (Visual) | 자막 (Text Overlay) | 내레이션/자막 텍스트 (Copy) |
| :--- | :--- | :--- | :--- |
| **0s-3s** | 🗣️ 입모양 애니메이션 + 소리<br>"이 소리는?" | `👄 shi`<br>`시? 치? 지?` | **"중국어 '시 (shì)'를 이렇게 발음하세요!"** |
| **3s-15s** | 🔊 발음 가이드 + 혀 위치<br>혀 모양 시각화 | `혀 끝을 잇고 입술 살짝`<br>`[ʂ] 소리가 나요` | **"혀 끝을 잇지 말고, 입술은 살짝 둡니다.<br>'시 (shì)'는 '치'보다 훨씬 부드러운 소리예요."** |
| **15s-20s** | 🎧 귀 이미지 + 단어 예시<br>"시장" 예시 | `시장`<br>`Shìchǎng` | **"이 발음으로 '시장 (Shìchǎng)'을 말해보세요.<br>정확한 발음을 연습하세요!"** |
| **CTA** | 👉 발음 팁 구독<br>"다음엔 성조 4 개!" | `🗣️ 정확한 발음 습관` | **"올바른 발음으로 중국어 실력 업그레이드! 👇"** |

---

### 🟤 유형 E: 상황극/대사 학습형 (Scenario & Dialogue)
**컨셉**: 실제 생활 상황을 시뮬레이션하여 회화 능력 향상.
**적용 시나리오**: 식당, 택시, 쇼핑 등

| 시간대 | 화면 (Visual) | 자막 (Text Overlay) | 내레이션/자막 텍스트 (Copy) |
| :--- | :--- | :--- | :--- |
| **0s-3s** | 🚖 택시 상황 + 표정<br>"택시 기사에게?" | `🚖 택시기사`<br>`"아니요!"` | **"중국인 택시기사에게 '아니요'라고 말해야 할 때!"** |
| **3s-15s** | 🗣️ 대사 카드 + 발음 가이드<br>"가르주세요" 예시 | `나오 (nǎi)`<br>`(술집/식당 등)` | **"중국인 식당에서 '술'을 주문할 때:<br>'나오 (nǎi)'라고 말하세요. (술집 이름)"** |
| **15s-20s** | 🇨🇳 중국 거리 + 실제 상황<br>"아니요" 표현 예시 | `안녕히 계세요`<br>`Niàn hǎo` | **"중국어로 '안녕히 계세요'는 '니앙 하오'(Niàn hǎo) 입니다.<br>다음엔 어떤 상황에서 쓰나요?"** |
| **CTA** | 👉 상황별 중국어 구독<br>"다음엔 식당 주문!" | `🍴 식당 중국어 실전` | **"실전에 필요한 중국어를 배워보세요! 👇"** |

## 3. 핵심 카피라이팅 가이드라인 (Tone & Voice)

| 구분 | 가이드라인 |
| :--- | :--- |
| **톤앤매너** | 친근하고, 명확하며, 동기 부여가 되는 톤<br>(예: "이거 꼭 보세요!", "다들 틀려요!") |
| **문장 길이** | 짧고 명료한 문장 (5~8 단어로)<br>자막은 한 줄에 1 문장 권장 |
| **감성** | 학습 성취감 강조<br>(예: "이 표현이면 중국인 친구가 됩니다!") |
| **금기** | 복잡한 설명, 긴 문장, 부정적인 표현 ("안됩니다" 대신 "이렇게 하면 좋습니다") |
