## 1. Executive Summary (요약)
**"Automated Chinese Language Education Shorts for Vietnamese Market"**  
본 기획안은 비개발자 크리에이터가 **자동 생성기(AutoShorts Engine)** 를 기반으로 중국어 교육 쇼츠 콘텐츠를 생산·배포하는 비즈니스 모델을 다룹니다. 핵심 전략은 **'고품질 콘텐츠 자동화'**를 통해 시간과 비용을 절감하고, 이를 통해 발생하는 수익 (AdSense, 디지털 상품 판매) 과 AI API 비용 간의 균형을 최적화하여 과금을 최소화하는 구조를 설계합니다.

## 2. 비즈니스 모델 (Business Model)

### 🏗️ 핵심 가치 제안 (Value Proposition)
**"AI 가 만드는 완벽한 중국어 교육 쇼츠"**  
- **고객**: 베트남인 중국어 학습자, 해외 거주 한국인 등 언어 학습을 원하는 타깃.
- **문제 해결**: 수동 콘텐츠 제작의 시간/비용 문제를 자동 생성기로 해결하고, 일관된 고품질 콘텐츠를 제공.

### 💰 수익화 모델 (Monetization Strategies)
**모델 A: AdSense 기반 (콘텐츠 조회수 수익)**  
- YouTube Shorts 광고 수익으로 주력.
- **목표**: CTR > 4%, 평균 시청 지속 시간 > 26 초 달성 시, 고수익 창출 가능.

**모델 B: 디지털 상품 판매 (High-Ticket Offer)**  
- $27 가이드 모델 ($27~$49 USD) 과 연계한 유료 콘텐츠 제공.
- **예시**: "중국어 회화 필수 100 문장 PDF", "중국어 발음 교정 오디오 파일".

**모델 C: 구독형 서비스 (Subscription)**  
- 월 $5~$10 에 맞춤형 학습 콘텐츠 및 AI 피드백 제공.

### 🛡️ 과금 최소화 구조 (Cost Optimization)
AI API 비용 증가를 막기 위한 **3 단계 로컬화 전략**.

| 단계 | 전략 | 실행 방법 | 기대 효과 |
| :--- | :--- | :--- | :--- |
| **1 차** | **로컬 AI 엔진 활용** | FLIX 등 로컬 AI 모델 설치, API 호출 최소화. | API 비용 0원 달성 (초기 투자비용 제외). |
| **2 차** | **캐싱 및 재사용** | 이미 생성된 스크립트/이미지 캐싱하여 중복 생성 방지. | API 호출 횟수 50% 감소. |
| **3 차** | **API 비용 분산** | 무료 티어 (OpenAI 등) 적극 활용, 유료 플랜은 고가용성 작업에만 제한. | 고정 비용 최소화. |

> **주의**: PayPal 매출 분석 도구 실패로 인해 현재 API 과금 위험을 인지하고 있습니다. 따라서 로컬 환경 구축이 시급합니다.

## 3. 초기 시장 진입 전략 (Market Entry)

### 🎯 타깃 시장 및 진입 방식
- **주요 시장**: 베트남 (중국어 학습 수요 증가), 한국 (해외 거주자).
- **진입 방식**: YouTube Shorts → Instagram Reels → TikTok 크로스 플랫폼 배포.

### 📊 초기 시장 진입 KPI (Key Performance Indicators)

| 지표 | 목표 수치 | 측정 시기 | 성공 기준 | 비고 |
| :--- | :--- | :--- | :--- | :--- |
| **CTR (클릭률)** | 4% 이상 | 첫 달 내 | AdSense 수익화 시작 시점 | 감정적 고통 (Pain Point) 구간 최적화 필요. |
| **Retention Rate** | 70% 이상 | 첫 달 내 | 평균 시청 지속 시간 > 26 초 | 후킹 (Hook) 및 학습지점 최적화. |
| **API 비용/월** | $50 미만 | 첫 달 내 | 로컬 AI 활용도 90% 달성 | 과금 위험 회피. |
| **단일 사용자 LTV** | $30 이상 | 3 월 차 | 가이드 모델 ($27) 판매율 > 10% | 수익화 모델 다각화. |
