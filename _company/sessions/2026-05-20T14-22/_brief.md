# 📋 작업 브리프

**원 명령:** 숏폼 디자인 템플릿 완료되면 png 형식으로 내가 결과물을 확인 할 수잇도록 해줘

## 요약
사용자가 숏폼 디자인 결과물을 PNG 이미지 형태로 받고 싶어 합니다. 하지만 현재 프로젝트의 디자인은 단순 이미지가 아닌, 애니메이션과 인터랙션이 핵심인 기술적 MVP 프로토타입이기 때문에 PNG로 전달할 수 없습니다. 기대치를 관리하고 필요한 아티팩트 형식으로 안내해야 합니다.

## 분배
- **🎨 Designer**: 사용자에게 현 단계의 디자인 산출물은 단순 정지 이미지(PNG)가 아닌, '상호작용 애니메이션'과 '기술적 명세서'가 핵심임을 다시 한번 설명하십시오. 현재 진행된 템플릿 작업물을 PNG로 보여주기 어렵다는 점을 설명하며, 대신 Figma 링크를 통한 인터랙티브 프로토타입 미리보기나, 움직이는 모바일 목업 비디오(Mock-up Video) 형태로 결과를 확인하는 방안을 제시하여 사용자 기대치를 관리하십시오.
