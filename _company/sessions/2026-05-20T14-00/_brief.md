# 📋 작업 브리프

**원 명령:** 디자이너에게 스티치 aPi 저장햇어 확인하고 숏폼 디자인 템플릿 보여줘

## 요약
디자이너가 저장한 '스티치 API'의 기술적 사양을 검토하고, 이 API를 활용하여 최종적인 숏폼 영상 디자인 템플릿 세트를 제작하는 것이 목표입니다. 시각 자료와 개발 통합 계획이 동시에 필요합니다.

## 분배
- **🎨 Designer**: 스티치 API 연동을 전제로 한 모바일 최적화(9:16 비율) 숏폼 영상 디자인 템플릿 3종 세트를 제작하십시오. 이 템플릿들은 뉴트럴 어스톤/빈티지 필름 질감을 유지해야 하며, API가 출력할 데이터가 들어갈 'Placeholder 영역'을 명확하게 지정하고, 해당 영역의 애니메이션 타이밍 가이드라인(예: 데이터 로딩 시 스피너, 정보 노출 시 페이드 인)까지 포함한 기술 브리프를 작성해주세요.
- **💻 코다리**: 디자이너가 활용할 '스티치 API'의 정확한 연동 가이드를 제공하십시오. 특히 숏폼 영상 콘텐츠에 맞춰, 어떤 데이터(예: 사용자 ID, 특정 키워드 데이터 등)를 어떤 방식으로 전송하고 받아와야 하는지 기술 명세서 형태로 작성해주세요. 이 API 통합 지점이 기존 랜딩 페이지 스켈레톤 코드 어디에 삽입되어야 가장 효과적인 자동화 루프가 될지 코멘트를 달아주세요.
- **✍️ Writer**: 새로 제작될 숏폼 디자인 템플릿 세트와 연동하여, '정보 습득 욕구'를 자극하는 강력한 후크(Hook) 스크립트 아이디어 5가지와 각 템플릿에 맞는 카피라이팅/CTA 문구를 작성해주세요. 이 카피는 짧은 시간 안에 시청자의 공감대를 얻고 다음 행동(구매/체험)으로 유도하는 데 초점을 맞춰야 합니다.
