# 📋 작업 브리프

**원 명령:** 쇼츠 영상 자동화를 위해 필요한 내용을 리스트업해서 알려줘

## 요약
쇼츠 영상 자동화 시스템 구축에 필요한 핵심 구성요소와 프로세스를 리스트업하는 기획 단계입니다. 콘텐츠, 디자인, 기술 세 영역에서 요구되는 구체적인 산출물 목록을 정의해야 합니다.

## 분배
- **✍️ Writer**: 자동화 쇼츠 영상의 표준 스크립트 템플릿과 핵심 카피라이팅 가이드를 작성하라. 주제별 후크(Hook) 문구, 학습 지점 설명 구조, 그리고 콜투액션(CTA)을 포함하는 마스터 스크립트를 정의해야 한다.
- **🎨 Designer**: 자동화 시스템에 필요한 모든 시각 에셋 컴포넌트 목록과 디자인 규격을 정리하라. 템플릿 기반으로 작동할 수 있도록 폰트, 컬러 팔레트, 레이아웃 그리드 등의 '디자인 시스템 가이드'를 최우선으로 확정해야 한다.
- **💻 코다리**: 자동화 워크플로우의 기술 아키텍처를 설계하라. 스크립트(Writer), 디자인 에셋(Designer)을 입력받아 최종 비디오 파일을 출력하기 위해 필요한 API 연동 지점, 코딩 모듈 목록, 그리고 로컬 기반 자동 실행에 최적화된 시스템 구조(코드 뼈대)를 상세히 리스트업해야 한다.
