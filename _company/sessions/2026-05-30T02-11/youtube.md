# 🎬 Ep.1: 시스템 오류 발견 (최종 스크립트 초안)

## 1. 영상 기본 정보
| 구분 | 내용 |
| :--- | :--- |
| **제목** | 더 이상 '느낌'에 의존하지 마세요. 시스템으로 증명하세요. |
| **시청자** | 중국어를 배우는 베트남인, 해외 거주 중인 한국인 (자가학습 선호층) |
| **핵심 메시지** | "관찰 → 추출 → 자동화 실행" 프로세스를 통해 학습 효율을 10 배로 늘립니다. |
| **CTA 시점** | 영상 내 **10~15 초** 구간 (Pain Point 극대화 후 전환) |
| **CTA 문구** | "단 12 개월로 시간 벌기 ($27)" |

## 2. 목표 시청자 정의 (Target Audience)
- **베트남인:** 중국어 비즈니스 의사소통 필요, 자기주도 학습 선호.
- **해외 거주 한국인:** 현지 언어 습득 및 효율적인 학습 도구 탐색 중.
- **공통 특성:** 기존 교재/강의 방식에 지쳐 있고, '시스템'을 통한 빠른 성취를 원함.

## 3. 스크립트 타임라인 & 구조 (검증된 지식 기반)

| 시간대 | 구분 | 비주얼/화면 구성 | 오디오/내레이션 | CTA 트리거 포인트 |
| :--- | :--- | :--- | :--- | :--- |
| **0:00 - 2s** | **[후킹]** | 📉 복잡한 다이어그램 → 빨간색 경고 텍스트 "시스템 오류" (1 초 깜빡임) | *(BGM 시작: 급박한 타이밍)*<br>"매일 이렇게 시간 버리는 거, 아시죠? 월 50 시간. 그걸 돈으로 환산하면..." | 🔥 **Hook:** 당신의 하루 24 시간은 '이거'에 쓰이고 있습니다. (손글씨 느낌) |
| **2s - 10s** | **[문제 제기]** | 🤯 "노동 vs 지능" 비교 이미지 (왼쪽: 손으로 처리 오류, 오른쪽: 정돈된 데이터) | "아직도 이걸 손으로 처리하고 계세요? 그건 '노동'입니다."<br>"우리가 필요한 건, '지능형 에이전트의 사고방식'입니다." | **Authority:** 기존 방식 부정 및 차원 상향 암시. |
| **10s - 15s** | **[해결책 제시]** | ⚡ **OEA 프로세스** 시각화 (관찰 → 추출 → 자동화) | "핵심은 세 단계입니다."<br>"첫째. 관찰(Observe). 무엇이 *문제*인지 파악하는 것부터 시작합니다."<br>"하지만 이게 다가 아니에요. 진짜 비밀은... '패턴 추출'입니다." | **CTA:** **[단 12 개월로 시간 벌기 ($27)]** (강제 노출) |
| **15s - 30s** | **[본론]** | 🔄 단계별 애니메이션 (필터링 효과, 키워드 강조) | "패턴을 추출(Extract)하고, 최종적으로 자동화 실행(Automate)."<br>"이 세 단계가 당신의 시간을 무한대로 복구시킬 겁니다." | **Retention:** 정보 밀도 및 보상 즉시 제공. |
| **30s - 45s** | **[결론]** | ✅ 밝고 깨끗한 화면, 흰색/베이지 자막 (정답 느낌) | "이 원리만 이해하면, 당신은 이미 에이전트 전문가입니다."<br>"다음 편 예고: 'A 를 B 로 바꾼 진짜 법'." | **Action:** 구독/알림 설정 버튼 강제 노출. |
