# 📚 Jinny`s Lab SEI 교육 상품 스토리텔링 마스터 가이드 (V1.0)

## 1. 프로그램 개요 (Product Overview)
- **프로그램명:** SEI (System Error Intervention) - "시스템 오류를 고치는 비법"
- **슬로건:** "당신의 시간은 더 이상 낭비되지 않는다."
- **핵심 가치:** 관찰 → 추출 → 자동화 (OEA 프로세스) 를 통한 시간 효율성 극대화
- **타겟:** 중국어 학습을 원하는 베트남인, 해외 거주 한국인 등

---

## 2. 세일즈 교육 상품 커리큘럼 & 핵심 카피라이팅 (Step-by-Step Copywriting)

### 🎬 Module 0: Before/After 비교 이미지 (Hook Stage)
**[비주얼]** 시스템 오류 아이콘 + 시간 낭비 통계 그래프
**[핵심 카피]:**
- **Headline:** "당신의 시간이 지금 얼마나 손실되고 있나요?"
- **Sub Copy:** "평균적으로 하루 3.2시간을 무의미한 반복 작업에 소비합니다."
- **CTA:** "시스템 오류를 고치는 법, 지금 시작하세요."

### 🧠 Module 1: 관찰 (Observe) - 문제 인식 단계
**[비주얼]** 사용자 행동 분석 도메인 + 데이터 시각화
**[핵심 카피]:**
- **Headline:** "무의식적인 습관이 당신의 시간을 훔칩니다."
- **Sub Copy:** "작은 반복 행동들이 쌓여 큰 낭비로 이어집니다. 이제 그걸 찾아내세요."
- **CTA:** "자신의 시간 사용 패턴을 분석해보세요 →"

### 🧬 Module 2: 추출 (Extract) - 솔루션 발견 단계
**[비주얼]** AI 알고리즘 흐름도 + 효율성 향상 그래프
**[핵심 카피]:**
- **Headline:** "낭비를 없애는 법, 시스템화하세요."
- **Sub Copy:** "수작업을 시스템으로 바꿔보세요. 10 배의 효율을 경험하세요."
- **CTA:** "시스템 전환 가이드를 무료로 받으세요 →"

### 🚀 Module 3: 자동화 (Automate) - 결과 달성 단계
**[비주얼]** 자동화 툴 인터페이스 + 시간 자유도 아이콘
**[핵심 카피]:**
- **Headline:** "자유로운 시간을 되찾으세요."
- **Sub Copy:** "자동화 툴을 활용하여 일상의 부하를 줄이세요. 더 많은 일에 집중하세요."
- **CTA:** "자동화 툴 체험 시작하기 →"

| **Module 0** | 시스템 오류 인식 | - 시간 낭비 사례 식별 능력<br>- 문제 상황 정의력 향상 | - 시간 낭비 사례 수: 5건 이상 식별<br>- 문제 정의 정확도: 85% 이상 |

| **Module 3** | 자동화 (Automate) | - 자동화 툴 활용 능력<br>- 결과물 완성도 | - 자동화 툴 사용 실습 완료<br>- 시간 절약 효과 측정: 30% 이상 |

## 4. 상품 스토리텔링 통합 지도 (Product Storytelling Map)

### 📍 Stage 0: Before/After 비교 이미지
- **목표:** 시청자의 '시간 낭비 후회' 감정 자극
- **배치 위치:** YouTube 쇼츠 초반 3 초 / 인스타그램 리얼스 오프닝
- **핵심 메시지:** "당신의 시간이 지금 얼마나 손실되고 있나요?"

### 📍 Stage 1: 관찰 (Observe) 단계
- **목표:** 문제 상황 인식 및 공감대 형성
- **배치 위치:** YouTube 쇼츠 중간 15~30 초 / 인스타그램 피드 중반부
- **핵심 메시지:** "작은 반복 행동들이 쌓여 큰 낭비로 이어집니다."

### 📍 Stage 2: 추출 (Extract) 단계
- **목표:** 솔루션 발견 및 기대감 조성
- **배치 위치:** YouTube 쇼츠 후반 30~45 초 / 인스타그램 피드 하단부
- **핵심 메시지:** "수작업을 시스템으로 바꿔보세요. 10 배의 효율을 경험하세요."

### 📍 Stage 3: 자동화 (Automate) 단계
- **목표:** 실제 체험 유도 및 구매 전환
- **배치 위치:** YouTube 쇼츠 마지막 CTA / 인스타그램 피드 하단부
- **핵심 메시지:** "자동화 툴 체험 시작하기 →"

## 6. CTA 배치 전략 (Call-to-Action Placement Strategy)

### 📍 YouTube 쇼츠
- **0~3 초:** Hook 카피 + Before/After 이미지 노출
- **15~30 초:** 관찰 단계 강조 후 "분석해보세요" CTA
- **30~45 초:** 자동화 단계 강조 후 "$27 체험 시작" CTA

### 📍 인스타그램 피드
- **오프닝:** Hook 카피 + Before/After 이미지
- **중간부:** 관찰/추출 단계별 핵심 메시지 노출
- **하단부:** "자동화 툴 체험 시작하기" CTA 버튼 배치
