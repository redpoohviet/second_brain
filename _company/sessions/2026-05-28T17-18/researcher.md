---

## 1. 타겟층 트렌드 분석 (Vietnamese Learners)

### 1.1 숏폼 콘텐츠 선호 형식
- **교육용 숏폼 특징**: 빠른 템포, 시각적 힌트 (문자, 발음 가이드), 실제 상황극 (Roleplay).
- **시청 지속율**: 시작 3 초 내 훅(Hook) 필수. "중국어 배우는 방법", "오류 시나리오" 등 구체적인 해결책 제시.
- **타겟 반응 요소**: 베트남인 학습자들은 '한국어-중국어' 비교 (비교 언어학적 접근) 에 높은 관심을 보임.

### 1.2 성공적인 교육 콘텐츠 포맷
- **포맷 A: 오답 시나리오** ("이 표현은 틀려요! 정답은?") — 클릭률과 학습 효과를 동시에 잡음.
- **포맷 B: 상황극 + 텍스트 오버레이** (예: "중국어 속담"을 일상 대화로 풀이) — 이해도 향상.
- **포맷 C: 시스템 오류/비효율성 강조** ("당신은 지금 시간 낭비하고 있어요") — 긴장감 유발하여 시청 유도.

### 1.3 핵심 니즈 도출 (Needs Extraction)
1. **시각적 피드백**: 텍스트 오버레이와 발음 가이드가 동시 제공되어야 함.
2. **즉각적인 활용성**: 이론보다는 실제 대화에 바로 쓸 수 있는 문장 중심.
3. **시스템 오류 강조**: 학습자의 기존 습관 (시간 낭비, 비효율) 을 지적하여 동기 부여.

---

## 2. 경쟁사 분석 (Competitor Analysis)

### 2.1 주요 성공 채널 유형
- **유형 A: 언어 스터디 채널**  
  - 특징: 일상 대화 중심, 빠른 템포, 짧은 영상.  
  - 트렌드: "30 초에 배우는 중국어", "오답 시나리오"가 인기.
- **유형 B: 시스템/교육 컨설팅 채널**  
  - 특징: 학습자의 비효율성을 지적하고 AI 도구나 방법론 제시.  
  - 트렌드: "당신의 시간 낭비, 이렇게 해결하세요" 등 강렬한 제목.

### 2.2 최근 성공 콘텐츠 사례 (가상 데이터 기반)
- **콘텐츠 1**: "중국어 회화 실수 Top 5 — 시스템 오류 시나리오"  
  - 조회수: 10 만+  
  - 특징: 오답을 강조하고 시각적 오버레이로 정답 제시.
- **콘텐츠 2**: "베트남인 중국어 학습자必看 — 한국어를 활용한 비교법"  
  - 조회수: 5 만+  
  - 특징: 언어학적 접근으로 흥미 유발.

### 2.3 경쟁사 미비점 (Opportunity)
- 대부분의 채널이 '이론'에 치중하여 실제 활용도가 낮음.
- 시각적 피드백 (오버레이, 텍스트) 이 부족한 콘텐츠가 많음.
- 시스템 오류/비효율성 강조를 통한 동기 부여가 부족함.

### 3.3 기대 효과

# 🚀 코딩 프로그램 기획안: 노출형 쇼츠 자동화 파이프라인

**프로젝트명:** Exposure Shorts Pipeline (ESP)  
**버전:** 1.0  
**연구자:** 🔍 Researcher  
**목표:** 베트남인 중국어 학습자를 타겟팅한 노출형 쇼츠 영상을 로컬 기반으로 자동 제작 및 최적화.

---

## 1. 프로그램 개요

### 1.1 배경
- 베트남인 중국어 학습자들은 **시각적 피드백**(텍스트 오버레이, 발음 가이드) 과 **즉각적인 활용성**을 중시함.  
- 기존 콘텐츠는 이론에 치중되어 있어 실제 활용도가 낮으며, 시스템 오류/비효율성 강조가 부족함.  
- 로컬 기반 자동화 파이프라인 구축으로 API 과금 최소화 및 안정성 확보 필요.

### 1.2 목표
- 트렌드 분석 데이터를 기반으로 콘텐츠를 자동으로 기획하고 제작.  
- 시각적 오버레이 (빨강 톤) 를 적용하여 학습자 타겟팅 강화.  
- YouTube 업로드 준비 상태 파일 (.md, .json) 생성 후 비공개 테스트 업로드.

---

## 2. 프로그램 구조 및 기능

### 2.1 입력 모듈 (Input Module)
- **트렌드 데이터**: 베트남인 중국어 학습자 트렌드, 경쟁사 분석 결과.  
- **기획서**: 콘텐츠 기획안 (제목, 썸네일, 설명).  
- **로컬 리소스**: 영상素材, 이미지, 오디오 파일.

### 2.2 처리 모듈 (Processing Module)
- **콘텐츠 기획 자동화**:  
  - 제목 생성: 트렌드 키워드 + 시스템 오류 강조.  
  - 썸네일 디자인: 빨강 톤 오버레이 적용, 텍스트 간결성 유지.  
  - 설명 작성: SEO 최적화 (베트남인 학습자 타겟팅).  
- **영상 편집 자동화**:  
  - 텍스트 오버레이 추가 (발음 가이드, 핵심 문장).  
  - 시각적 피드백 요소 (오답 시나리오, 시스템 오류 강조) 적용.

### 2.3 출력 모듈 (Output Module)
- **업로드 준비 파일**: `.md`, `.json` 형식으로 생성.  
- **비공개 테스트 업로드**: YouTube Studio 연동 또는 로컬 대체 프로세스.

---

## 3. 개발 일정 및 우선순위

### 3.1 단계별 목표
1. **입력 모듈 구현**: 트렌드 데이터 및 기획서 파싱.  
2. **처리 모듈 구현**: 콘텐츠 자동화 (제목, 썸네일, 영상 편집).  
3. **출력 모듈 구현**: 업로드 준비 파일 생성 및 테스트 업로드.

### 3.2 우선순위
1. **로컬 기반 자동화 파이프라인 구축** (API 과금 최소화).  
2. **시각적 오버레이 (빨강 톤) 적용** (학습자 타겟팅 강화).  
3. **비공개 테스트 업로드** (프로세스 검증).

---

## 4. 기대 효과

- **비용 절감**: 로컬 기반 자동화로 API 과금 최소화.  
- **효율성 향상**: 트렌드 데이터 기반 콘텐츠 기획으로 생산성 극대화.  
- **타겟팅 강화**: 시각적 피드백과 시스템 오류 강조를 통한 학습자 동기 부여.
