# 🎬 [Final Script] System Error: The Language Learning Glitch

**프로젝트명:** AutoShorts En (시스템 오류 진단: 언어 학습 편)  
**타겟:** 베트남인 / 해외 거주 한국인 (중국어 교육 필요)  
**톤앤매너:** 네온/글리치 비주얼 기반, 진지하지만 몰입감 있는 '시스템 관리자' 톤  
**총 분량:** 40~50 초 (숏폼 최적)

---

## 📋 스크립트 구조 (Hook / Body / CTA)

### **[0s ~ 3s] Hook: 시스템 오류 메시지 (System Error Message)**
- **시각:** 화면 중앙에 붉은색 네온 글리치 텍스트 "SYSTEM ERROR: TIME_WASTED"
- **내레이션 (Voiceover):** "당신의 언어 학습 시간, 90% 가 이 '시스템 오류' 때문에 사라지고 있습니다."
- **자막:** `ERROR 404: TIME NOT FOUND` (네온 색상 #FF3366)

### **[3s ~ 15s] Pain Point Max (문제의 증폭)**
- **시각:** 배경이 노란색 경고 신호 (#ffaa00) 로 점멸하며, 불필요한 학습 자료들이 화면을 가르는 글리치 효과.
- **내레이션:** "단편적인 단어 암기만 반복하다 보면, 뇌는 새로운 언어를 '파싱'하지 못합니다. 결과? 계속 같은 실수를 반복하고 시간 낭비."
- **자막 (동작):** 
  - `Observe:` (관찰) → `Extract:` (추출) → `Automate:` (자동화) 순서로 화면에 네온 텍스트가 떠오름.

### **[15s ~ 30s] Solution 제시 (OEA 프로세스 적용)**
- **시각:** 글리치 노이즈가 사라지고, 깔끔한 흰색/네온 파란색 UI 로 전환. 학습 자료들이 자동으로 정렬되는 애니메이션.
- **내레이션:** "저는 이를 '시스템 오류'라고 정의합니다. 해결책은 단순 암기가 아닌 **'OEA 프로세스'** 입니다. 관찰 → 추출 → 자동화. 비효율을 시스템으로 대체하세요."
- **자막 (동작):** `SYSTEM FIXED` (초록색 네온)

### **[30s ~ 40s] CTA (Call to Action)**
- **시각:** 화면 하단에 채널 로고와 "무료 가이드 다운로드" 버튼이 팝업으로 나타남.
- **내레이션:** "시간을 아끼는 첫걸음은 '오류 진단'부터입니다. 지금 바로 구독하세요."
- **자막 (동작):** 
  - `Subscribe` (구독)
  - `Download Guide` (가이드 다운로드)
