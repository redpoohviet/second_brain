## 🎬 YouTube Shorts 스크립트 초안: 'OEA 프로세스' 최종 버전
*(테마: 시간 낭비에 대한 공감적 후회(Regret) 자극 및 시스템 도입의 압도적 가치 제시)*
*(길이: 약 38초 - 알고리즘 최적화 분량)*

---

### 💡 스크립트 목표 및 구조 분석
*   **Pain Point:** 현재 비효율적인 수동 작업 방식에 대한 시간적 후회와 스트레스.
*   **핵심 메시지:** '관찰(O) → 추출(E) → 자동화(A)'의 시스템만이 이 문제를 근본적으로 해결한다.
*   **톤앤매너:** 초반 3초는 충격/공포 (Pain), 중반은 해방감/지식 전달 (Solution), 후반은 긴박함/기회 포착 (CTA).

### ⏱️ 상세 스크립트 및 지시사항

| Timecode | 내레이션 (Voice Over) | 자막 / 화면 구성 (Visual Cue) | Tone & BGM / 효과음 (Audio Cue) |
| :---: | :--- | :--- | :--- |
| **[0s - 3s]** | **(HOOK)** "혹시 아직도, 이 데이터 하나를 정리하는 데 '몇 시간'을 쓰고 계신가요?" | **화면:** 극단적으로 빠른 화면 전환. (스프레드시트 오류 메시지, 복붙되는 커서 깜빡임 등) / **자막 강조:** 🔴 *시간 낭비 중* | **BGM:** 긴장감 최고조의 드럼 비트 시작. / **SFX:** '딩-' 하는 알람 소리 + 빠르고 불안한 배경음악 (높은 볼륨). |
| **[4s - 8s]** | "매번 수동으로 데이터를 정리하고, 오류를 수정하느라... 밤을 새우는 경험." | **화면:** 지친 얼굴의 사람(배우) 클로즈업. 책상 위 쌓인 커피잔과 자료들. '수작업'이라는 단어에 빨간색 X 표시. / **자막:** ❌ 수동 작업 = 시간 폭탄 | **BGM:** 긴장감이 잠시 풀리지만, 여전히 낮은 불안감을 유지하는 서브 베이스 노이즈. |
| **[9s - 12s]** | "이게 바로 '비효율성'입니다. 시간만 태우는 과정이죠." | **화면:** 복잡한 플로우차트가 엉키고, 한 곳에서 폭발하는 애니메이션 (시각적 혼란). / **자막:** 비효율성의 증거: 데이터의 홍수 | **BGM:** '삐익-' 하는 고주파 경고음. 문제 제기 시 명확한 톤앤매너 변화. |
| **[13s - 20s]** | "하지만 이제, 그 과정을 완전히 시스템으로 만들 수 있습니다. 이게 바로 OEA 프로세스입니다." | **화면:** 화면이 깨끗하게 정리되면서 'O' (Observe) → 'E' (Extract) → 'A' (Automate) 순서로 깔끔한 흐름도가 나타남. / **자막:** 💡 OEA: 정보 처리의 새 기준 | **BGM:** 경쾌하고 명확하며, 자신감을 주는 멜로디로 급변. SFX: '클릭-' 하는 시스템 작동음. |
| **[21s - 28s]** | "어떤 데이터를 *관찰*하고(O), 핵심을 *추출*해서(E), 최종적으로 *자동화*하는(A) 단 하나의 플로우만 아시면 됩니다." | **화면:** OEA 단계별로 실제 데이터가 매끄럽게 변환되는 시뮬레이션 영상. (예: 비정형 텍스트 → 구조화된 표) / **자막:** 관찰(O) > 추출(E) > 자동화(A). 시스템은 거짓말을 안 합니다. | **BGM:** 성공과 진보를 상징하는 웅장하고 리드미컬한 사운드. (가장 높은 몰입도 구간). |
| **[29s - 34s]** | **(CTA)** "이 시스템, 남들 모르는 '정보처리 비밀 무기'입니다. 지금 이 영상을 보는 순간에도 당신은 시간 낭비를 하고 있을지 모릅니다." | **화면:** 화면 중앙에 $27 상품 이미지와 'Limited Offer' 배지가 박힌다. 긴장감을 주는 타이머 카운트다운 애니메이션. / **자막 강조 (🚨):** 🔥 *이걸 모른다면, 시간만 버립니다!* | **BGM:** 갑자기 볼륨을 낮추고 긴박한 목소리로 속삭이는 듯한 톤. SFX: '틱톡' 하는 카운트다운 소리 극대화. |
| **[35s - 38s]** | "지금, OEA 프로세스를 마스터하고 당신의 시간을 되찾으세요. 프로필 링크를 확인하세요." | **화면:** 상품 구매 버튼(CTA)이 화면 전체에 걸쳐 크게 표시됨. (뉴트럴 어스톤 필름 질감 유지). / **자막:** ⬇️ 시간 절약, 지금 바로 시작! [링크] | **BGM:** 강력한 코다(Coda)로 마무리되며 희망적이고 명료하게 끝남. |

### 📌 스크립트 구현 가이드라인 요약 (Technical Spec for Video Production)

1.  **시각 자산 일관성:** 모든 배경과 UI 요소는 뉴트럴 어스톤/빈티지 필름 질감을 유지하며, 오직 OEA 플로우가 등장할 때만 애니메이션 효과를 최대화합니다.
2.  **사운드 디자인 (Sound Design):** 후크(0-3s) 구간의 불안한 BGM과 CTA(29s-) 구간의 긴박한 BGM 대비가 가장 중요합니다. 사운드가 감정적 페인 포인트를 증폭시켜야 합니다.
3.  **데이터 레이블 변수화:** 스크립트 내 모든 핵심 문구("시간 낭비", "OEA 프로세스", "이걸 모른다면 시간만 버립니다!")는 단순 자막이 아니라, 별도의 **[Variable: PAIN_POINT]**, **[Variable: OEA_DEFINITION]** 등의 데이터 레이블 변수로 분리하여 코딩해야 합니다.
