# 📋 작업 브리프

**원 명령:** [기획안] AI 기반 숏폼(Shorts) 영상 제작 완전 자동화 시스템
프로젝트명: AutoShorts Engine (가칭)
기획 목적: 대본 작성, 이미지 생성, 비디오 변환, TTS 음성 합성, 최종 영상 편집 및 자막 싱크 합치기 등 기존의 수작업 쇼츠 제작 프로세스(약 4시간 소요)를 단 5단계의 코드로 완전 자동화하여 시간 및 비용(영상 편당 0~500원 이하)을 혁신적으로 절감하는 시스템 구축.

1. 영상 핵심 프로세스 분석 (벤치마킹)
대상 영상에서 분석한 쇼츠 자동화의 핵심 파이프라인과 3대 기술 원리는 다음과 같습니다.

📌 5단계 자동화 파이프라인
대본 생성: AI 프롬프트를 통해 숏폼에 최적화된 대본 및 장면별 이미지 묘사(프롬프트) 동시 생성

이미지 생성: 대본의 각 장면에 맞는 이미지를 일관된 스타일(그림체)로 생성

영상 변환(Image-to-Video): 정지 이미지를 역동적인 고화질 비디오 클립으로 변환 (타사 프로그램과의 차별성)

음성(TTS) 및 자막 매칭: 텍스트를 음성으로 변환하고 밀리초(ms) 단위로 자막 싱크 계산

최종 편집: 비디오, 오디오, 자막, 폰트를 하나로 합성하여 완성본 출력

📌 3대 핵심 기술 원리
CLI & API 활용: 웹사이트 UI에 접속할 필요 없이 터미널과 API를 통해 AI 모델(Gemini, Claude 등) 직접 호출

브라우저 자동화 (Browser Automation): 공식 API가 없는 무료 AI 서비스의 경우, 스크립트가 크롬 브라우저를 직접 제어하여 이미지 업로드 및 다운로드 수행 (비용 절감 핵심)

FFmpeg 코딩 편집: 영상 편집 툴(CapCut, Premiere 등) 없이 코드를 통해 영상 자르기, 합치기, 자막 렌더링 처리

2. 시스템 아키텍처 및 상세 기능 설계
본 기획안은 Python 환경을 기반으로 하며, 영상에서 언급된 도구와 비용 최적화 솔루션을 반영합니다.

🛠 단계별 모듈 상세 설계
1단계: Script & Prompt Generator (대본 및 프롬프트 생성)
기능: 유저가 '주제(예: 깨두부의 효능)'를 입력하면 쇼츠 문법(어그로 오프닝 -> 본론 -> 댓글 유도)에 맞는 대본을 생성하고, 각 문장에 맞는 이미지 생성용 영어 프롬프트를 함께 출력합니다.

사용 기술: Gemini API (영상 기준 가입 시 40만 원 상당의 무료 크레딧 제공으로 비용 최적화)

산출물: script.json (문장 내용, 자막 텍스트, 이미지 프롬프트 매핑)

2단계: AI Image Batch Generator (이미지 배치 생성)
기능: 1단계에서 생성된 7~10개의 장면별 프롬프트를 바탕으로 이미지를 자동 생성합니다. 영상 전체의 일관성을 위해 프롬프트 앞에 지정된 스타일 문구(예: Cinematic HD, Minecraft style, Vector illustration 등)를 강제로 접두어로 붙입니다.

사용 기술: Pollinations AI API (완전 무료) 또는 Gemini API

산출물: scene_01.png ~ scene_07.png

3단계: Image-to-Video Converter (이미지 애니메이션화)
기능: 멈춰있는 이미지를 2~3초짜리 움직이는 고품질 숏폼 비디오 클립으로 변환합니다. 정지 화면에 확대 효과만 주는 기존 방식과 차별화되는 고퀄리티 요인입니다.

사용 기술: Grok.com/img2vid (무료 서비스) + Playwright 또는 Selenium (브라우저 자동화 라이브러리)

자동화 로직: 코드가 자동으로 크롬을 열어 Grok에 접속 -> 이미지 업로드 -> 스타일 유지를 위해 Maintain visual style 프롬프트 입력 -> 생성된 영상 자동 다운로드.

산출물: scene_01.mp4 ~ scene_07.mp4

4단계: TTS & Sync Calculator (음성 합성 및 자막 타이밍 계산)
기능: 대본을 자연스러운 AI 성우 목소리로 변환하고, 자막이 오디오와 완벽히 일치하도록 밀리초(ms) 단위의 싱크 파일(SRT 등)을 생성합니다.

사용 기술: Google TTS (기본/무료), 고품질 필요 시 ElevenLabs API 또는 Supertone API 연동.

산출물: narration.mp3, subtitles.srt (또는 밀리초 단위 오디오 길이 데이터)

5단계: FFmpeg Video Mixer (최종 영상 편집기)
기능: 변환된 비디오 클립들을 순서대로 이어 붙이고, 내레이션 오디오를 입힌 후, 지정된 예쁜 폰트와 스타일을 적용해 자막을 화면 중앙에 렌더링합니다.

사용 기술: FFmpeg (Python의 ffmpeg-python 또는 MoviePy 라이브러리 활용)

추가 기능: 시스템 내에 유행하는 폰트 10종(가독성 높은 서체, 레트로 서체 등)을 기본 내장하여 선택 가능하도록 구현.

산출물: final_shorts.mp4

## 요약
AI 기반 숏폼 영상 제작 자동화 시스템(AutoShorts Engine) 구축을 위한 초기 기획 및 핵심 스타일 가이드라인을 설정합니다.
