# 📺 썸네일 전략 브리프 (V1.0) — CTR 4% 목표 달성 가이드

## 1. 목적
Designer 와 Developer 가 완성한 `CTA_Tech_Spec_V1.0.md` 에셋을 실제로 클릭으로 연결하는 **썸네일 전략**을 수립합니다. 비개발자 크리에이터의 특성상, 복잡한 디자인보다 **'즉각적 자극' (Shock Value)** 과 **'사회적 증명'** 이 CTR 달성에 결정적입니다.

## 2. 핵심 원칙 (KPI 기반)
- **CTR 목표**: 4% 이상 (현재 평균 대비 15% 상승 필요). `[근거: 회사 정체성 - 디자인 에셋은 설정된 KPI(CTR 4% 이상)]`
- **Drop-off Rate**: 15% 감소 (썸네일 클릭 후 영상 이탈 방지).
- **시각적 메시지**: 3 초 이내 뇌로 인식 가능한 단순한 텍스트/이미지.

## 3. 썸네일 전략 상세 (버전 A/B)

### 버전 A: 사회적 증명 + 극단적 비교 (강력 추천)
- **컨셉**: "남들은 저렇게 하고, 나는 이렇게 합니다." (Before/After 시각화).
- **텍스트**: 🔥 **"월 50시간 버리는 법!"** 또는 **"이거만 하면 끝."** (긴 글 금지).
- **비주얼**:
  - Background: 어두운 배경 (주의력 집중).
  - Foreground: 복잡한 데이터 그래프 vs 깔끔한 자동화 시스템 비교. `[근거: 레오 검증된 지식 - Self-RAG 지식 - Before/After 시각화]`
  - CTA 에셋: `cta-primary` 컴포넌트 사용 (`scale(1.02)` + `box-shadow` 효과).

### 버전 B: 권위 빌드업 (Authority Build-up)
- **컨셉**: "이 원리만 알면, 당신은 이미 전문가입니다."
- **텍스트**: 🧠 **"AI 자동화의 핵심 원리"** 또는 **"3 단계로 시작하세요."**
- **비주얼**:
  - Background: 밝고 깨끗한 배경 (정답의 느낌). `[근거: 레오 검증된 지식 - Self-RAG 지식 - 화면은 밝고 깨끗하게 정리하고]`
  - Foreground: '관찰 → 추출 → 자동화' 프로세스 도표.
