# 🎓 온라인 강의 커리큘럼 목차 초안 (V1.0)
## 🚀 "AI 1인 기업: 단순 자동화를 넘어 '지능형 비즈니스'로"

> **강좌 목표:** 비개발자, 비전공자가 AI 자동화 도구를 활용해 실제 수익을 창출하는 지능형 비즈니스 구축 방법 전수  
> **대상 학습자:** 중국어/베트남어 교육 콘텐츠 크리에이터, 유튜브 쇼츠 제작자, 소규모 사업주  
> **교육 기간:** 총 5 모듈 (각 모듈 2~3 시간) + 최종 프로젝트  

---

## 📑 커리큘럼 구조

### 🎯 Module 0: 입문 및 마인드셋 (1 시간)
| 시분 | 주제 | 핵심 메시지 | CTA |
|------|------|-------------|-----|
| 0-3s | **후크:** "당신의 시간이 왜 사라지는지 알고 있나요?" | 시각적 타임랩스 + 숫자 애니메이션 | 👉 "지금부터 시작해보세요" |
| 3-15s | **문제 정의:** 수동 작업의 비용과 기회비용 | 실제 사례 통계 제시 | ✅ "이 강의에서 해결합니다" |
| 15-30s | **해결책 소개:** AI 자동화 파이프라인 | 시스템 아키텍처 시각화 | 🔗 "Module 1 으로 이동하기" |

**핵심 콘텐츠:**
- 지능형 비즈니스의 정의 (단순 자동화 vs 지능형)
- 비개발자가 왜 AI 자동화를 배워야 하는지
- 학습 목표 및 성공 기준 설정
- **CTA:** 레오/코다리 에이전트 연동 테스트 환경 확인

### 💼 Module 1: 수익화 전략과 비즈니스 모델 설계 (2 시간)
| 시분 | 주제 | 핵심 메시지 | CTA |

| 5-20s | **비즈니스 모델 분석:** 제휴 마케팅 vs 직접 판매 | 수익 구조 비교 차트 | ✅ "당신의 모델은 무엇인가요?" |
| 20-35s | **가격 전략:** $27 진입 장벽 모델의 이점/단점 | 실제 사례 데이터 | 🔗 "Module 2 로 진행하기" |

**핵심 콘텐츠:**
- 수익화 KPI 설정 (조회수 → 전환 → 수익)

- 가격 책정 기준과 진입 장벽 설계
- 판매 페이지 UX/UI 기본 원칙
- **CTA:** Designer 와 판매 페이지 디자인 검토 요청

### 🔧 Module 2: 자동화 파이프라인 구축 기초 (2 시간)
| 시분 | 주제 | 핵심 메시지 | CTA |

| 0-5s | **후크:** "코드 한 줄도 필요 없어요" | 무코딩 도구 시각적 데모 | 👉 "자동화 시작하는 법을 알려드릴게요" |
| 5-20s | **핵심 프로세스:** 관찰 → 추출 → 자동화 (OEA) | 플로우차트 애니메이션 | ✅ "3 단계로 분해됩니다" |
| 20-35s | **도구 소개:** 주요 AI 자동화 도구 연동법 | 툴별 아이콘 + 기능 설명 | 🔗 "실습 환경 준비하기" |

**핵심 콘텐츠:**
- 파일 시스템 경로 안정성 확보 방법

- 코다리(코딩 에이전트) 연동 기초
- **CTA:** 코드다리 테스트 스크립트 실행 가이드 확인
