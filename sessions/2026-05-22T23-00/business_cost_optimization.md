## 1. 현황 분석 및 핵심 리스크 식별
현재 시스템이 직면한 가장 큰 비용 발생원은 **외부 AI API 호출량**입니다.
- **리스크 포인트:** 영상이 길어질수록, 텍스트/이미지 생성 횟수가 늘어날수록 과금 비용은 선형적으로 증가합니다.
- **데이터 부족:** 현재 실시간 API 로그가 없으므로, 실제 비용을 정확히 파악할 수 없습니다. **(자가검증: 추측 0 개 / 근거 1 개)**

## 2. 비용 효율성 극대화 전략 (AI Cost Reduction)

### A. API 호출 최소화 로직 도입 (Local Caching & Prompt Optimization)
1.  **로컬 캐싱 (Local Vector Cache):**
    - 시스템이 같은 키워드나 유사한 요청을 반복적으로 API 에 보내는 것을 방지합니다.
    - **실행:** Redis 또는 SQLite 기반의 간단한 캐싱 레이어를 추가하여, 중복된 프롬프트 분석은 로컬에서 수행하도록 변경합니다.
2.  **프롬프트 압축 (Prompt Compression):**
    - 불필요한 토큰 사용량을 줄여 API 비용 절감 및 속도 향상.
    - **실행:** `AutoShorts`의 프롬프트 템플릿을 정밀 분석하여, 최소한의 토큰으로 최대의 효과를 내도록 재설계합니다.

### B. 하이브리드 모델 아키텍처 (Hybrid Model Architecture)
- **기본형 (Standard):** 일반적인 요청은 로컬에서 구동 가능한 경량 모델 (예: Llama-3-8B 또는 Mistral) 을 우선 사용합니다. 비용 0원 가능.
- **프리미엄형 (Premium):** 로컬 모델이 처리 불가능한 복잡한 요청만 고비용 API 를 호출하도록 라우팅합니다.

## 3. 수익화 모델 최적화 (Monetization Strategy)

### A. 'AI 비용'을 '가치'로 전환하는 Tier 구조
사용자가 AI 비용을 아끼려 하는 것과, 우리 시스템이 제공하는 '고급 기능'을 연결합니다.
- **Basic Plan:** 로컬 캐싱 기반, 표준 프롬프트 사용 (저비용). → **가격: $9**
- **Pro Plan:** 고급 API 모델 사용, 개인화 프롬프트 추가. → **가격: $27 (현재 목표)**
- **Enterprise:** 전체 비용 청구서 제공, API 로그 분석 리포트. → **가격: $49+**

### B. 마이크로 퍼널과 CTA 최적화
- **CTA 시점:** 사용자가 "시간 낭비"라는 감성적 고통을 느낀 직후 (T=10~15s) 에, AI 비용 절감을 강조한 CTA 를 배치합니다.
  - 예시 카피: *"시스템 오류 진단 비용까지 아끼세요. $27 로 시작하는 자동화 솔루션"*
