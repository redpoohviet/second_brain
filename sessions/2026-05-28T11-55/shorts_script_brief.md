## 📋 1. 스크립트 구조 (0~25 초)

### [0s-3s] 후킹 (Hook)
```
[화면: 빨간 텍스트 오버레이 + 큰 중국어 자막]
🇻🇳 "베트남인! 이걸 모르면 중국어로 1 년 낭비했습니다!"

[VO/자막]: 
"베트남 친구들이 저에게 가장 많이 묻는 질문이에요."
"저도 처음엔 중국어 공부한 지 3 개월 만에 실생활에서 못 썼다고 좌절했죠."
```

### [3s-8s] 학습지점 (Pain Point)
```
[화면: 중국어 책 페이지 + 베트남인 얼굴 오버레이]
🇻🇳 "베트남 친구가: '왜 중국어 공부했는데 못 써요?'"

[VO/자막]: 
"문법만 외우고 있어요."
"단어를 알지만 말을 못해요."
"실생활에서 활용되지 않는 공부가 시간 낭비예요."
```

### [8s-15s] 실생활 활용 구조 (Solution)
```
[화면: 중국어 점포 + 주문 장면 + 베트남인과의 대화]
🇻🇳 "베트남 친구가: '이제야 알겠어요!' 하는 표정"

[VO/자막]: 
"저는 이렇게 가르쳤어요:"
"1. 매일 5 분 실전 회화 연습"
"2. 중국어 점포에서 주문하기"
"3. 베트남 친구들과 중화권 문화 비교하며 배우기"
```

### [15s-20s] CTA (Call to Action)
```
[화면: 구독 버튼 + 체크 표시 애니메이션]
🇻🇳 "베트남 친구! 지금 시작하세요!"

[VO/자막]: 
"중국어를 진짜 쓰려면 이 방법을 따라하세요."
"구독하고 다음 영상에서 더 배우세요!"
```
