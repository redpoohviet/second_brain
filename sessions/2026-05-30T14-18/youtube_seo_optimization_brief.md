## 🎯 SEO 최적화 전략 (검증된 지식 기반)

### 1. 제목 최적화 공식
```
[痛點] + [해결책] + [CTA] = 클릭 유도 제목

예시: "하루 50시간 버리는 당신을 위한 AI 자동화 법칙 🤖"
     → "Time Wastage Pain Point" + "AI Solution" + "Curiosity CTA"
```

### 2. 해시태그 전략 (Shorts 알고리즘 최적화)
```yaml
주요 태그 (5개): #아자동화 #시간절약 #생산성
서브 태그 (10개): #인공지능 #효율성 #비즈니스팁 #자기계발 #AI툴
롱테일 태그 (3개): #중국어배우기 #한국어공부 #언어학습
```

### 3. 설명란 최적화 (검색 노출)
```
[1-2줄] 영상 핵심 가치 (키워드 포함)
[3-5줄] 타임스탬프/주제별 링크
[마지막] CTA: "구독/알림 설정 버튼 누르세요!"
```

### 4. 업로드 시간대 (KST 기준)
| 요일 | 최적 시간 | 이유 |
|------|----------|------|
| 평일 | 12:30-13:30 / 19:30-21:00 | 직장인 피크타임 |
| 주말 | 10:00-12:00 | 자유시간 활용 |
